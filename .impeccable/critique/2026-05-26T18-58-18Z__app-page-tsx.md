## Anti-patterns verdict

**LLM assessment.** First-order reflex (dark + neon for a sports product) still survives but is no longer doing the work alone. The dark glass vocabulary is internally consistent now, gradient text is gone, the FeaturedNews light-card outlier is dark. The second-order test improved: this no longer reads as a stock sportsbook clone. The remaining sportsbook tells are now isolated to copy ("Bets lock", "$240 pot", "+25 pts earned"), not visual treatment.

**Deterministic scan.** Local detect.mjs engine bundle still missing; fell back to npx -y impeccable@2.1.9 detect --json on app/page.tsx, app/layout.tsx, and all of components/landing/. Result: [] zero findings. The 3 gradient-text warnings the prior critique caught are all resolved. Direct grep also confirms: no bg-clip-text anywhere in components/landing/.

**Em-dash status.** Direct grep finds 19 across 6 files (was 61 across 13). Down 70%; the worst offenders (faq-section.tsx 28, pillar-section.tsx 8) were orphan files deleted in the distill pass. Still a DESIGN.md ban violation, just smaller.

**Visual overlays.** Unavailable, no browser MCP in this environment. Same as prior critique.

## Overall impression

The page has decisively cleared its two privacy-hypocrisy P0s and most of its visual-noise issues. What's left is language: the copy still reaches for sportsbook tells the visual system has stopped reaching for. The single biggest remaining opportunity is a copy pass to retire "Bets," "pot," and "+25 pts earned" the same way the quieter pass retired the rainbow accents.

## What's working

1. **The privacy promise is now true.** Vercel Analytics is gone; flag SVGs self-host; no third-party CDN calls anywhere on the landing.
2. **The dark vocabulary is internally consistent.** Glass cards share the same ground, inset highlight, hairline border. FeaturedNews now feels like the same product as the rest of the page.
3. **Three-Channel Rule actually enforced.** Mint = your actions, cyan = info/live, amber = leader. Features uses only mint + cyan. Stats numbers are white.
4. **Skip-link present and brand-correct.** Mint pill on focus, top-left, z-indexed above Navbar; main element receives focus.
