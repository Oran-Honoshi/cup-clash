## Priority issues

### [P1] Bunting strip runs forever, no prefers-reduced-motion guard
- Where: navbar.tsx @keyframes buntingScroll with animation: buntingScroll 28.8s linear infinite. No prefers-reduced-motion in the file.
- Why it matters: most prominent residual motion offender after the prior cleanup passes. WCAG 2.3.3 + PRODUCT.md accessibility commitment.
- Fix: @media (prefers-reduced-motion: reduce) { .animate-bunting { animation: none; } }.
- Suggested command: /impeccable harden

### [P2] FAQ single-open mode blocks comparison; Enterprise CTA opens a modal mid-funnel

- Suggested command: /impeccable polish for FAQ, /impeccable shape for Enterprise route migration.
